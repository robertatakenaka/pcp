====================================================
Bienvenido a la documentación de SciELO PC Programs!
====================================================

**SciELO PC Programs** es un conjunto de herramientas para producir entrada de datos para el proceso de publicación de artículos en una colección SciELO.


Para la **Producción de XML**:

- **XML Package Maker**

  - valida pacotes de XML SPS producidos por cualesquier herramientas. 
  - **uso obrigatório** para la producción de pacotes XML validados de acuerdo con SPS.

- **Markup**

  - macro para Word para identificar los elementos estructurales y semánticos dentro de un texto.
  - generar el XML SPS a partir de la identificación de estes elementos.
  - **uso opcional** pues es permitido e incentivado a usar cualesquier herramientas para producir los XML de acuerdo con el SPS.


Para **Gestão de Coleções** (para uso **somente** de las Coordenaciones de una Colección SciELO):

- Title Manager
- Converter
- XML Converter
- XML SciELO (Exportador de XML para otras bases: ISI y PubMed)
